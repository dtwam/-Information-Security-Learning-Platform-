
# 🛡️ Information Security Learning Platform  
## منصة تعلم الأمن السيبراني

منصة تعليمية تفاعلية لتعلم أساسيات الأمن السيبراني والاختبار الأمني بشكل عملي وآمن.

An interactive educational platform for learning cybersecurity fundamentals and security testing in a safe, practical environment.
